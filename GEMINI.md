# Gemini Project Analysis

## Project Overview

This project is a simple web-based calculator. It uses HTML for the structure, CSS for styling, and JavaScript for the calculator's logic.

## Project Structure

*   `index.html`: The main HTML file that defines the structure of the calculator.
*   `style.css`: The CSS file that styles the calculator's appearance.
*   `script.js`: The JavaScript file that implements the calculator's functionality.

## Building and Running

To run the calculator, simply open the `index.html` file in a web browser.

There are no specific development conventions established for this project yet.
